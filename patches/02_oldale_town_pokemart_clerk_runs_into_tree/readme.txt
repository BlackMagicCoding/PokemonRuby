Fixed the bug with the PokeMart clerk, if you have a full inventory and speak to them again.
A detailed explanation can be seen here: https://youtu.be/FNVHavDAlbY

Either use the .ips patch on a Vanilla ROM, or the .rbc script, which will fix the glitch and dynamically move the new code to a free offset in your ROM (thus working with other ROM Hacks).
